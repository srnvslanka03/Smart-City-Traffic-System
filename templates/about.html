Our Mission

The Smart City Traffic System is an innovative platform designed to revolutionize urban traffic management through adaptive signal control and computer vision technology. Our system dynamically adjusts traffic light timings based on real-time vehicle detection and density analysis, significantly reducing congestion and improving traffic flow in cities.

Existing Systems vs Proposed Model

Existing Fixed-Time Signals

  • Static cycle lengths regardless of demand fluctuations.
  • Manual retiming needed; slow to respond to incidents/events.
  • Limited visibility into per-lane performance.
  • Higher average delay and idling emissions.

Our Adaptive Proposed Model

  • Real-time vision counts and type classification drive signal timing.
  • Dynamic green splits and phase sequencing per-second.
  • Lane-level analytics, throughput and wait-time dashboards.
  • Lower delay, improved throughput, and reduced emissions.

YOLO Model Comparison

Our system supports multiple YOLO (You Only Look Once) models for vehicle detection. Here's a comparison of the available options:

YOLOv12

  • Pros: State-of-the-art accuracy, advanced transformer-based architecture, superior performance in complex scenes
  • Cons: Highest computational requirements, larger model size
  • Use Case: Cutting-edge applications requiring maximum precision and robustness
  • Size: ~50MB

YOLOv5

  • Pros: Fast inference, lightweight, good accuracy
  • Cons: Slightly lower precision than newer versions
  • Use Case: Real-time applications with limited resources
  • Size: ~27MB

YOLOv8

  • Pros: High accuracy, improved architecture
  • Cons: Moderate computational requirements
  • Use Case: High-precision detection in complex environments
  • Size: ~43MB

YOLOv7

  • Pros: Balanced performance and accuracy
  • Cons: Moderate resource usage
  • Use Case: General-purpose deployment
  • Size: ~36MB

YOLOv6

  • Pros: Efficient on edge devices
  • Cons: Lower accuracy in crowded scenes
  • Use Case: IoT and embedded systems
  • Size: ~22MB

Current default: YOLOv12 for state-of-the-art accuracy and performance

How It Works

The Smart City Traffic System leverages computer vision algorithms to detect vehicles at intersections in real-time. Using cameras and image processing, it identifies vehicle types and counts, feeding this data into adaptive algorithms that adjust traffic light timings dynamically. This ensures optimal flow by prioritizing busy lanes and reducing unnecessary waiting times.
